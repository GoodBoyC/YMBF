PREMIUM FACEBOOK CLONING TOOL



# How to install
## Setup
```bash
pkg update
pkg upgrade
pkg install python
pkg install python2
pip install requests
pip install mechanize
pip install lolcat
pip install bs4
pip install rich
pip install nodejs
pip install cython
pkg install git
rm -rf brutefb
```

## install clone
```bash
git clone https://github.com/GoodboyC/brutefb
```

## Run tool
```bash
cd brutefb
python run.py
```

## Update tool
```bash
cd brutefb
git pull
python run.py
```
